# Olly Henson Coaching — Meditation Download Funnel

A custom-coded landing page funnel for GHL, built section by section in HTML/CSS.

## Quick Start
1. Read `ghl-setup-guide.md` for full GHL setup instructions
2. Paste `custom-css.css` contents into GHL Custom CSS field
3. Add each `sections/0X-*.html` file into its own GHL Custom HTML block, in order
4. Set all section margins to 0 in GHL
5. Add and hide the native GHL form (see setup guide)
6. Replace all placeholder copy with final copy
7. Update redirect URL in the hidden GHL form settings

## Files
| File | Purpose |
