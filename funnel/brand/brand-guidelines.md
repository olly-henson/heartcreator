# Olly Henson Coaching — Brand Guidelines

## Colour Palette

| Name | Hex | Use |
|------|-----|-----|
| Space Black | #080010 | Primary background |
| Deep Purple | #1a0535 | Secondary background |
| Mid Purple | #6b21a8 | Accents, gradients |
| Violet | #a855f7 | Highlights, links |
| Magenta | #d946ef | Primary accent, CTAs |
| Pink Glow | #f0abfc | Light accents |
| Star White | #f8f4ff | Primary text |
| Muted Lavender | #c4b5fd | Secondary text |

## Gradients
- **Primary CTA:** `#d946ef` → `#6b21a8` (135deg)
- **Headline accent:** `#d946ef` → `#a855f7` (135deg)
- **Background nebula:** `#2d0a5e` → `#1a0535` → `#080010`

## Typography
- **Headings:** Playfair Display — Bold/Black (900)
- **Body:** Inter — Regular (400), Medium (500), SemiBold (600), Bold (700)
- Both available free on Google Fonts

## Heading Style
- Main headline: Playfair Display 900, white
- Italic accent line: Playfair Display italic, magenta→violet gradient
- Subheading: Inter 400, muted lavender (#c4b5fd)

## Logo / Brand Mark
- Text-based: "Olly Henson Coaching" in Playfair Display
- Accent colour: Magenta (#d946ef)

## Button Style
- Shape: Pill (border-radius: 50px)
- Background: Gradient #d946ef → #6b21a8
- Text: Inter Bold, white, 18px
- Shadow: 0 4px 30px rgba(217,70,239,0.4)

## Image Style
- Dark, cosmic, moody backgrounds
- Warm human subjects with energy/light effects
- Purple/magenta colour grading overlay

- YouTube Thumbnail: 1280 x 720px
- YouTube Banner: 2560 x 1440px
