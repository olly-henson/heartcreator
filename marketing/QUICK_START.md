# Quick Start — Olly Henson Coaching AI Operating System

> Five minutes from download to your first AI-powered output.

---

## What You've Got

Your AI operating system is a set of markdown files that teach any AI assistant to work like you — your voice, your audience, your rules.

The most important file is **AGENT.md** — your business brain. Load it first, and the AI immediately knows who you are, who you serve, and how you sound.

---

## Setup by Platform

### Claude (claude.ai)

1. Start a new conversation
2. Click the paperclip icon to attach files
3. Upload **AGENT.md**
4. Optionally upload any skill files you need (e.g. `skills/brand-voice.md`)
5. Start prompting — Claude now knows your business

**Pro tip:** Create a Project in Claude and add all your files there. Every conversation in that Project will have your full OS loaded.

### Claude Code (CLI)

1. Copy **AGENT.md** to your project root as `CLAUDE.md`
2. Place the `skills/` folder in the same directory
3. Claude Code automatically reads CLAUDE.md at the start of every session
4. Reference skill files naturally: "Use my brand voice guide to write a LinkedIn post"

### ChatGPT

1. Go to **Explore GPTs** > **Create a GPT**
2. In the **Configure** tab, paste the contents of **AGENT.md** into the **Instructions** field
3. Under **Knowledge**, upload your skill files
4. Save and start chatting with your custom GPT

**Alternative:** In any chat, click the paperclip and upload AGENT.md as context.

### Google Gemini

1. Go to **Gemini** > **Gems** > **Create a Gem**
2. Paste the contents of **AGENT.md** into the Gem's instructions
3. Optionally paste skill file contents as additional context
4. Save and start using your Gem

### Microsoft Copilot

1. Open **Copilot** > **Agents** (or equivalent)
2. Create a new Agent
3. Paste the contents of **AGENT.md** into the agent configuration
4. Upload skill files as knowledge sources
5. Save and start using

## Your First Prompt

Once AGENT.md is loaded, try this:

> "Write me a LinkedIn post about [pick a topic from your content pillars]. Use my brand voice."

The AI should produce something that sounds like CEO Olly Henson wrote it — not generic AI slop. If it doesn't, check that AGENT.md is properly loaded.

## Loading Skill Files

For specific tasks, tell the AI to reference a skill file:

- "Using my content strategy, plan next week's content"
- "Using my brand voice guide, write an Instagram caption for this photo"
- "Using my content formulas, write 3 hooks for a carousel about [topic]"
