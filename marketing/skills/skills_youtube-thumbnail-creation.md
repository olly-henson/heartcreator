# YouTube Thumbnail Creation — Olly Henson Coaching

> This skill generates a thumbnail concept for any YouTube video. Read the video entry, then produce four things: background, expression, overlay image, and text overlay words. Olly builds it in Canva from there.

---

## How to Generate a Thumbnail Concept

When Olly asks for a thumbnail, read the video's Title, Type of Post and Hook from Notion, then output a brief covering these four elements only:

1. **Background** — which background and why
2. **Expression** — which expression category and what it should convey
3. **Overlay image** — what the secondary image should show and where it sits
4. **Text overlay** — 2–5 words, emphasis treatment if applicable

Present it in plain language. No Canva steps. No folder references. Just the creative plan.

---

## Established Layout (from live thumbnails)

**Default layout:** Face right (takes up roughly half the frame), overlay image top-left, text bottom-centre or bottom-left. This is the consistent pattern across all published thumbnails — use it unless there's a strong reason not to.

**The overlay image** sits in the top-left zone with a soft circular or feathered crop. This is a deliberate style choice — it's part of the visual identity, not a mistake. Always suggest an overlay image unless the concept is so strong the face alone carries it.

**The magenta arrow** is an optional connecting element — points from the overlay image toward Olly's face. Use it when the overlay and face have a cause-and-effect relationship (e.g. "this is the problem" → "I have the answer"). Don't use it on every thumbnail.

**Text position varies by layout:**
- Bottom — default when overlay is top-left and face is right
- Top — use when the overlay image is large and central (e.g. anatomical heart filling the left half), freeing the middle and bottom for visuals

**Overlay crop style varies:**
- Soft circular/feathered crop — default for people/scene images
- No crop, blend directly into background — use for graphic/illustrated overlays like the glowing heart anatomy image. Looks more seamless than a circle on non-photographic assets

## Background Options

Background assets live in the Backgrounds folder in Canva. Reference specific assets from there when available. If the folder is empty, specify the colour or describe what's needed.

| Background | When to use |
|------------|------------|
| Space Black `#080010` | Default. Works for almost everything |
| Deep Purple `#1a0535` | Slightly warmer — good for split-text layouts |
| Stars PNG / cosmic | Mission content, expansive concept videos — e.g. "We Need You" used a full cosmic galaxy background |

## Expression Categories

Match to the energy of the video — the expression should feel like the punchline of the hook, not a posed photo.

| Expression | What it looks like | Best for |
|------------|--------------------|----------|
| **Quiet Certainty** | Calm, settled, looking slightly past camera — "I know something you don't" | Heart science, mechanism, feel-it-first angles |
| **Mid-Thought Intensity** | Slight lean, looking off to one side, mid-sentence energy | Argument-based, "why it isn't working" angles |
| **Pattern Interrupt** | Direct to camera, one eyebrow slightly raised | Curiosity hooks, surprising facts |
| **Relaxed Confidence** | Slight smirk, grounded, unhurried — pointing gesture works here | "You're doing it wrong", "still in your head" angles |
| **Soft Vulnerability** | Eyes slightly down or to the side, introspective | Personal story, chronic illness thread |
| **Mission Energy** | Direct to camera, pointing outward — purposeful, direct | Mission content, "We Need You" energy |
| **Genuine Surprise** | Eyes slightly wider, jaw slightly open — natural, not performed | Turning point moments |

## Overlay Images

The secondary image top-left is a core part of the design — it visualises the concept the video is about. Always suggest one unless the brief is purely face-led.

**Style:** Soft circular or feathered crop blending into the dark background. AI-generated images have worked well.

| Concept | Overlay image |
|---------|--------------|
| The feeling / elevated emotion | Woman holding glowing light orb, golden hour, peaceful expression |
| Stress / survival mode / stuck | Woman with head down or hands to temples, overwhelmed |
| Mission / collective awakening | Silhouette crowd, glowing heart ring, cosmic background |
| Heart coherence / the heart | Close crop of hand on chest, warm light |
| The subconscious / the mind | Brain graphic, neural network visual, or person deep in thought |
| The heart as mechanism | Anatomical glowing heart — blue and magenta tones, scientific style. No circular crop — blend directly into background |

**Magenta arrow:** Optional. Add when the overlay shows the problem and Olly's face is the solution — it creates a visual "because of this → here's the answer" flow.

## Text Overlay Rules

- **2–5 words** — short and punchy. The hook bank has examples
- Must feel like the tension or punchline of the video — not a description of it
- Creates curiosity without giving the answer away
- All caps preferred for punchy short hooks. Title case with ellipsis ("This is why...") works for curiosity-gap hooks
- **Emphasis treatment options:** underline a key word (e.g. "MAKE THE FEELING STICK" with FEELING underlined) OR use magenta on one word. Not both
- The text is not the video title — it scroll-stops, the title does the SEO

### Three Text Styles

**1. Curiosity-gap** — leaves something unresolved, drives the click
```
This is why... / Won't stick / The missing piece
```

**2. Declarative pointing** — text points directly at the overlay image as a reveal. Used when the overlay IS the answer or the problem. Works with the magenta arrow.
```
IT IS THIS
```
Why it works: the viewer looks at the overlay, looks at the text, looks back at Olly. Creates a visual argument in a single glance. Use when the overlay image shows the cause or the solution.

**3. Statement of truth** — a bold claim that creates curiosity without being a question
```
Feel it first / Not your mind / Your heart knows
```

### Deriving the Text From the Video

| Video type | Text approach |
|------------|--------------|
| Argument-Based | Name the failure state or use declarative pointing if the overlay shows the cause — "This is why...", "IT IS THIS", "Won't stick" |
| Educational | Name the surprising truth — "Your heart knows", "Feel it first", "Not your mind" |
| Personal Story | Name the turning point — "I went all in", "Then everything shifted" |
| Mission | Name the call — "We Need You", "Heart over head", "The world needs this" |
| Client Success | Name the transformation — "It actually worked", "She got there" |

---

## Never

- Flag or suggest removing the circular/feathered crop on overlay images — it is intentional visual identity, not a mistake. It is part of the established style across all published thumbnails
- Include Canva steps or production instructions in the brief — creative concept only
- Use both underline and magenta colour emphasis on the same thumbnail — pick one
- Use the cosmic/stars background for anything other than mission content
- Suggest more than one extra element beyond face and overlay — the power is in simplicity

---

## Brief Output Format

```
THUMBNAIL — [Video Title]

Background: [which one and why]
Expression: [category] — [one line on what it should convey]
Overlay image: [what to show, where it sits, crop style, magenta arrow yes/no]
Text overlay: [WORDS] — [emphasis treatment: underline / magenta word / none]
```
