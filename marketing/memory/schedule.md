## Recurring Tasks

### Content Production Cycle

```
Every week:
1. Create anchor content piece             [AUTO]
2. Review and approve                      [ASK]
3. Repurpose into social content           [AUTO]
4. Schedule across platforms               [ASK]
5. Monitor and engage                      [AUTO]
6. Log performance                         [AUTO]
```

### Engagement Cycle (Daily)

```
1. Check comments and DMs                 [AUTO]
2. Draft responses                         [AUTO]
3. Flag important conversations            [AUTO]
4. Send responses (if governance allows)   [ASK]
```
