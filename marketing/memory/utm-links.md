## How to read these links

Each link has three core tags:
- `utm_source` ‚Äî the platform (instagram, youtube)
- `utm_medium` ‚Äî how it got there (bio, reels, stories, description, pinned_comment)
- `utm_campaign` ‚Äî what it was promoting (meditation, coaching)

Internal funnel links use `ref=` instead of UTM (so they don't overwrite the original acquisition source).
