# Prospect Intelligence Log — Olly Henson Coaching

> A running log of real-world intel from clients, prospects, DMs, comments, and coaching calls. Raw captures only — no polish needed. This file feeds into `prospect_avatars.md` and `argument_sheet.md` over time, and informs all content generation.

---

## How to add intel

Olly starts any message with **"Intel:"** and pastes the raw information. The AI will:
1. Add it to this log with a date and source tag
2. Note which avatar(s) it relates to
3. Flag if it's strong enough to update an avatar or the argument sheet
4. Suggest a content angle if one is obvious
