# Business Management — Olly Henson Coaching

Daily operations hub. Sits across all business areas to keep priorities clear and work moving.

## How to use this project

1. Open Claude Code pointing at this folder
2. Claude reads `priorities.md` automatically and surfaces today's top priorities
3. Tell Claude what you want to work on — it will load the relevant skills/context files from the right project folder
4. When tasks are done, Claude updates `priorities.md`

## Key files

- `priorities.md` — live task list, read every session
- `CLAUDE.md` — context for Claude: business areas, rules, file paths
